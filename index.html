No-Resistors Wiring Logic

By enabling the internal pull-up resistors in Arduino code, we keep things incredibly clean. You only solder the pins directly to the switch, with a shared Ground (GND).

Arduino Pro Micro
Pin 2 (RX) Left Blinker Trigger
Pin 3 (TX) Right Blinker Trigger
Pin 4 High Beams (Pull flash)
Pin 5 Horn or Wiper push button
GND Common Ground (Solder to Switch Common)
How to map cheap Amazon/car stalks:

Universal stalks don't always ship with pin manuals. Don't worry!

  1. Set your Multimeter to the Continuity setting (has a wifi/soundwave icon; it beeps when probe tips touch).
  2. Touch one probe to a lead and toggle the stalk to the Left position. Keep testing other leads until the meter beeps.
  3. This gives you the Common (GND) and Left Pin.
  4. Keep the probe on the Common pin, switch stalk to the Right position, and probe other pins to locate the Right Pin.

Game Controller Firmware

Flashes on your Arduino Pro Micro to act as a native Plug-and-Play USB Gamepad.

#include 

// Define input pins
const int PIN_LEFT_BLINKER  = 2;
const int PIN_RIGHT_BLINKER = 3;
const int PIN_HIGH_BEAM     = 4;
const int PIN_HORN          = 5;

// Create standard Gamepad device with 4 button inputs
Joystick_ Joystick(
  JOYSTICK_DEFAULT_REPORT_ID, 
  JOYSTICK_TYPE_JOYPAD, 
  4, 0,                  // Button Count, Hat Switch Count
  false, false, false,   // No X, Y, or Z movement
  false, false, false,   // No rotary axes
  false, false,          // No rudder/throttle
  false, false, false    // No racing axes
);

void setup() {
  // Use internal pullup resistors - connects button output straight to pins
  pinMode(PIN_LEFT_BLINKER, INPUT_PULLUP);
  pinMode(PIN_RIGHT_BLINKER, INPUT_PULLUP);
  pinMode(PIN_HIGH_BEAM, INPUT_PULLUP);
  pinMode(PIN_HORN, INPUT_PULLUP);

  Joystick.begin();
}

void loop() {
  // Read pins and update buttons. Active low: digitalRead of LOW means button pressed.
  Joystick.setButton(0, !digitalRead(PIN_LEFT_BLINKER));
  Joystick.setButton(1, !digitalRead(PIN_RIGHT_BLINKER));
  Joystick.setButton(2, !digitalRead(PIN_HIGH_BEAM));
  Joystick.setButton(3, !digitalRead(PIN_HORN));

  delay(10); // Debounce
}

Option 1: Desktop G-Clamp

Best for players who disassemble their setups. Design a housing with a bottom 1/4" metal thread nut insert or a dual-wing clamp slot to lock down right next to your steering wheel on any desk.

Option 2: Wheelbase Spacer

Ideal for standard wheelbases (Moza, Fanatec). Create a slim circular plate that fits sandwiched between your wheelbase and the desk/rig mount. Extend an arm out to mount the stalk right behind the wheel rim.

Option 3: Aluminum Rig Profile

For high-end setups. Add standardized M6 mounting holes on the back of your chassis spaced 40mm apart. Lock it in with standard T-Nuts directly into your aluminum profile rig posts.

Design Tolerances Checklist

  • Measure the main shaft diameter of your replacement stalk switch using digital calipers. Stalk bases are typically ~22mm or ~25mm.
  • Include a built-in cable route and strain-relief channel for the Micro-USB cable inside the casing to prevent tearing during high-feedback wheel vibrations.
  • Maintain a clearance of 0.2mm for 3D printing tolerances on tight snapping components.

Simulator Setup Instructions

Because we've made this a standard USB gamepad, mapping couldn't be simpler!

Euro Truck / American Truck Sim

Open Options -> Controls, scroll down to controllers, and ensure your "Arduino Gamepad" is listed as an active device alongside your steering wheel. Go to Keys & Buttons and map Left Signal, Right Signal, High Beam, and Horn directly by triggering your physical switches.

Assetto Corsa (via Content Manager)

Under Settings -> Assetto Corsa -> Controls -> Buttons, scroll to Patch bindings (or CSP Custom bindings). Assign Left Signal and Right Signal to the registered Joystick button inputs.

BeamNG.drive

Go to Options -> Controls -> Bindings. Filter by Vehicle, click Toggle Left Signal and toggle your physical stalk left. It will bind instantly and work seamlessly on any vehicle in game!
